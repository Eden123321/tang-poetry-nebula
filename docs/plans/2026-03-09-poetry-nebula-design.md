# 诗歌意象3D星云可视化 - 产品设计文档

**版本**: MVP 1.0
**日期**: 2026-03-09
**状态**: 已批准

---

## 一、产品概述

创建一个可交互的3D星云可视化系统，展示唐诗中的意象及其关系。

## 二、技术栈

- **前端框架**: React 19 + Vite 7
- **3D可视化**: react-force-graph-3d (基于Three.js)
- **数据来源**: 开源唐诗数据集 (chinese-poetry)

## 三、核心功能

### 3.1 双层星云结构

**核心意象层（结构宇宙）**
- 50个核心意象节点
- 决定星云整体结构
- 大节点、高亮显示
- 显示星团之间连线

**具体意象层（细节层）**
- 围绕核心意象展开
- 点击核心节点时展开
- 小节点，局部显示

### 3.2 情绪颜色系统

| 情绪 | 颜色 |
|------|------|
| 思乡/离乡 | 蓝色 #4A90D9 |
| 离别/送别 | 金色 #D4A574 |
| 秋思/悲秋 | 紫色 #9B59B6 |
| 豪放/乐观 | 红色 #E74C3C |
| 清雅/闲适 | 青色 #1ABC9C |

### 3.3 交互设计

| 操作 | 行为 |
|------|------|
| 悬停 | 显示意象名称 |
| 点击 | 展开细节层 + 显示相关诗歌面板 |
| 拖拽 | 旋转3D视角 |
| 滚轮 | 缩放 |

## 四、视觉风格

- **背景**: 深空黑 #050510
- **效果**: 节点发光 (glow)
- **色调**: 霓虹情绪色
- **字体**: 标题宋体 + UI无衬线体

## 五、数据流程

```
开源唐诗JSON → 意象提取 → 共现统计 → 星云数据生成 → 3D渲染
```

## 六、MVP交付物

1. 唐诗语料数据 (JSON)
2. 50个核心意象定义
3. 意象映射词典
4. 意象提取与统计模块
5. 星云数据生成器
6. 3D可视化组件（含交互）
7. 诗歌详情面板
